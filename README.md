# itsYouke

itsYouke 是一个个人网站，用于发布和展示个人项目。

## 已有项目

### 幻兽帕鲁配种档案

幻兽帕鲁配种档案是一个用于记录和检索帕鲁配种结果的响应式网页工具，采用淡粉与淡蓝主题，适配手机和电脑端。

主要功能：

- 通过图鉴编号、中文名或英文名搜索并选择帕鲁。
- 记录“亲代 A + 亲代 B → 出生帕鲁”的配种结果。
- 按配种方案或出生结果搜索历史记录。
- 删除单条记录或清空全部记录。
- 生成与导入同步密钥，在不同浏览器或设备之间迁移进度。
- 所有记录默认保存在当前浏览器，不上传第三方服务。

访问路径：

```text
https://itsyouke.com/palbreed/
```

### 三角洲扑克牌收集

三角洲扑克牌收集是一个用于记录扑克牌收集进度的静态网页工具。

主要功能：

- 标记每张扑克牌的收集状态。
- 查看当前已收集数量和整体收集进度。
- 按花色、数字或未收集优先进行排序。
- 一键全选或清空收集状态。
- 导出进度密钥，用于在其他浏览器或设备中同步进度。
- 导入进度密钥，恢复已有收集状态。

访问路径：

```text
https://itsyouke.com/sjzpoker/
```
